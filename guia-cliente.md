# Guía rápida para mostrar al cliente

1. Abrir el link de GitHub Pages.
2. Mostrar la página de inicio.
3. Entrar a Catálogo y explicar que los modelos se cargan desde Supabase.
4. Entrar a Admin, subir un lente JPG/PNG y guardarlo.
5. Volver a Catálogo y mostrar que el lente aparece automáticamente.
6. Entrar a Probador AR, seleccionar el lente y activar la cámara.
7. Ajustar posición/tamaño si es necesario.
8. Capturar foto para demostrar el resultado.

Mensaje comercial sugerido:

“Esta plataforma permite que una óptica suba sus modelos de lentes y que sus clientes puedan probarlos desde la cámara del computador antes de comprar. Los diseños quedan guardados en la nube y el catálogo se actualiza automáticamente.”
